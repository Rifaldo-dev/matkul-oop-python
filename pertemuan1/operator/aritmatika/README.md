# Operator Aritmatika

Operator aritmatika digunakan pada nilai numerik untuk melakukan operasi matematika umum.

| Operator | Nama Operasi | Contoh |
|--------|-------------|-------|
| + | Penjumlahan | x + y |
| - | Pengurangan | x - y |
| * | Perkalian | x * y |
| / | Pembagian | x / y |
| % | Modulus (Sisa bagi) | x % y |
| ** | Perpangkatan | x ** y |
| // | Pembagian bulat (Floor division) | x // y |
